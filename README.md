## Project Description
Simple Forum is a decentralized message board built on the Flare (Coston2) blockchain.  
Users can create public posts that are permanently stored on-chain, providing a transparent, censorship-resistant messaging system.

The project demonstrates:
- How to interact with smart contracts using **Wagmi + Viem**
- How to perform **read/write operations**
- Wallet-gated UI interactions
- Real-time blockchain updates using transaction receipts

## How It Solves the Problem  
Traditional platforms store messages on centralized servers, meaning:
- Posts can be censored  
- Data can be modified  
- Ownership is unclear  

**Simple Forum** solves this by placing messages directly on the blockchain:  
- Posts cannot be deleted or altered  
- Users truly own their posts  
- A transparent and immutable public board exists for everyone
